# TsukiNote

Googleカレンダー風の月間カレンダーと、ノートアプリ風のメモを組み合わせたブラウザアプリです。

## 起動方法

`index.html` をダブルクリックしてブラウザで開いてください。インストールは不要です。

## 主な機能

- 月間カレンダーとミニカレンダー
- 予定の追加・編集・削除、時刻・カテゴリ・メモ
- 仕事／プライベート／学習の表示切り替え
- クイックメモ、メモ一覧、ピン留め
- 予定とメモの横断検索（Ctrl/Cmd + K）
- ダークモード
- LocalStorage による自動保存
- JSONファイルによるデータの書き出し・読み込み
- スマートフォン向けの下部ナビゲーション、サイドメニュー、セーフエリア対応

データは使用中のブラウザ内だけに保存されます。

## バックアップと端末間の移行

画面右上の「データ」を押し、「バックアップを書き出す」を選ぶと、予定・タスク・メモ・目標・運動記録・日記が1つのJSONファイルに保存されます。

別の端末では同じ画面から「バックアップを読み込む」を選び、書き出したJSONファイルを指定してください。読み込むと、その端末にある現在のTsukiNoteデータはバックアップの内容で置き換わります。

## スマートフォンでの操作

画面下のナビゲーションから「予定・メモ・目標・運動・日記」を切り替えられます。目標を押すと今月・今週の目標を確認・編集できます。カレンダー上部には締切の近いタスクが表示され、タップすると編集できます。左上のメニューボタンからはミニカレンダーとマイカレンダーを確認できます。予定画面右下の「＋」から予定またはタスクを追加できます。
