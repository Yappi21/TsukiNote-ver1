# TsukiNote

Googleカレンダー風の月間カレンダーと、ノートアプリ風のメモを組み合わせたブラウザアプリです。

## 起動方法

`index.html` をダブルクリックしてブラウザで開いてください。インストールは不要です。

## 主な機能

- 月間カレンダーとミニカレンダー
- 予定の追加・編集・削除、時刻・カテゴリ・メモ
- 仕事／プライベート／学習の表示切り替え
- クイックメモ、メモ一覧、ピン留め
- 予定とメモの横断検索（Ctrl/Cmd + K）
- ダークモード
- LocalStorage による自動保存
- JSONファイルによるデータの書き出し・読み込み
- OneDriveによるiPhone・パソコン間の自動同期
- スマートフォン向けの下部ナビゲーション、サイドメニュー、セーフエリア対応

データは使用中のブラウザ内に保存され、任意で個人用OneDriveと同期できます。

## バックアップと端末間の移行

画面右上の「データ」を押し、「バックアップを書き出す」を選ぶと、予定・タスク・メモ・目標・運動記録・日記が1つのJSONファイルに保存されます。

別の端末では同じ画面から「バックアップを読み込む」を選び、書き出したJSONファイルを指定してください。読み込むと、その端末にある現在のTsukiNoteデータはバックアップの内容で置き換わります。

## OneDrive同期

GitHub Pagesで公開したTsukiNoteを開き、右上の「データ」から「Microsoftでログイン」を選びます。初回の端末では、すでに予定などを入力している場合は「この端末 → OneDrive」、新しい端末では「OneDrive → この端末」を選んでください。

初回同期後は、TsukiNoteを開いている間に行った変更が約2秒後にOneDriveへ保存されます。別端末の変更は30秒ごと、画面へ戻ったとき、オンラインへ復帰したときに自動確認します。iPhoneでアプリを閉じている間のバックグラウンド同期は行われません。

同期データはOneDrive内の `アプリ/TsukiNote/TsukiNote-data.json` に保存されます。TsukiNoteに許可されるのは、この専用フォルダーの読み書きだけです。`index.html` を直接開いたローカル版ではMicrosoftログインを利用できません。

## スマートフォンでの操作

画面下のナビゲーションから「予定・メモ・目標・運動・日記」を切り替えられます。目標を押すと今月・今週の目標を確認・編集できます。カレンダー上部には締切の近いタスクが表示され、タップすると編集できます。左上のメニューボタンからはミニカレンダーとマイカレンダーを確認できます。予定画面右下の「＋」から予定またはタスクを追加できます。
